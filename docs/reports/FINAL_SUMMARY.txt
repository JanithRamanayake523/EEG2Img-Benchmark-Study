Models:

  • Vision Transformer (ViT-Tiny, ViT-Small, ViT-Base)
  • Lightweight CNN
  • 1D CNN, BiLSTM, Transformer (raw-signal baselines)
  • EEGNet (compact EEG-specific architecture)

Transformations:
  • Gramian Angular Fields (GASF/GADF)
  • Markov Transition Fields with quantization
  • Recurrence Plots with phase-space reconstruction
  • STFT-based Spectrograms
  • CWT-based Scalograms
  • Spatio-Spectral Feature Images (topographic maps)

Metrics & Tests:
  • Classification: Accuracy, Precision, Recall, F1, Cohen's Kappa, MCC, AUC
  • Statistical: Wilcoxon, t-test, ANOVA, post-hoc tests
  • Robustness: Noise injection, channel dropout, temporal shifts
  • Augmentation: MixUp, CutMix, geometric transforms

Training Features:
  • Mixed precision training (AMP)
  • Gradient accumulation
  • Early stopping and checkpointing
  • Learning rate scheduling
  • 5-fold and LOSO cross-validation
  • Data augmentation pipeline

════════════════════════════════════════════════════════════════════════════════
  USAGE QUICK START
════════════════════════════════════════════════════════════════════════════════

Clone Repository:
  $ git clone git@github.com:JanithRamanayake523/EEG2Img-Benchmark-Study.git
  $ cd EEG2Img-Benchmark-Study

Install Dependencies:
  $ python -m venv venv
  $ source venv/bin/activate  # On Windows: venv\Scripts\activate
  $ pip install -r requirements.txt

Run Single Experiment:
  $ python experiments/scripts/run_experiment.py \
      --config configs/experiment_baseline.yaml

Run Grid Search:
  $ python experiments/scripts/run_grid_search.py \
      --configs configs/*.yaml --output results/

Analyze Results:
  $ jupyter notebook notebooks/04_results_analysis.ipynb
