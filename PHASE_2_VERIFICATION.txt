HOW TO USE THE DATA

Python Code Example:

  import h5py
  
  # Open the combined HDF5 file
  with h5py.File('data/BCI_IV_2a.hdf5', 'r') as f:
      # List all subjects
      subjects = list(f.keys())  # ['subject_A01T', 'subject_A02T', ...]
      
      # Load data for one subject
      subject = 'subject_A01T'
      signals = f[f'{subject}/signals'][:]  # Shape: (240, 25, 751)
      labels = f[f'{subject}/labels'][:]    # Shape: (240,)
      
      # Access single trial
      trial = signals[0]                # Shape: (25, 751)
      trial_class = labels[0]           # Integer: 0-3
